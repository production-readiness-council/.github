# Ownership Structure

## Current state

The organization is in early setup. Repository ownership is currently assigned to `@rajan-sharma-in` as an interim owner through `CODEOWNERS`.

## Intended evolution

Ownership should move from a single user to explicit teams as governance matures.

Target team shape:

- maintainers: repository workflow and hygiene
- council: governance and publication authority
- working-group teams: domain-scoped operational ownership

## Rule

Single-user ownership should be treated as temporary bootstrapping, not the desired steady state for governance-critical repositories.

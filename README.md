# Production Readiness Council GitHub Defaults

This repository holds the organization-wide community health files and default collaboration rules for the Production Readiness Council.

These defaults are intended to support a neutral, evidence-based, open contribution model.

Primary references:

- `CONTRIBUTING.md` for contribution workflow expectations
- `CODE_OF_CONDUCT.md` for participant conduct
- `SECURITY.md` for vulnerability reporting and disclosure handling
- `pull_request_template.md` for change submissions
